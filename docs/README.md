# AI Copilot для контакт-центру — Технічна архітектура

> **Контекст**: Нова Пошта, контакт-центр, 1,000 операторів, 30,000 дзвінків/день, власна система РМ КЦ (CRM)
> **Телефонна платформа**: Cisco UCCE 12.6(2) + CUCM 12.6 + CVP 12.6 + Cisco Finesse 12.6

---

## Пріоритет впровадження

| Ітерація | Компоненти | Мета |
|---|---|---|
| **Ітерація 1 ⭐** | Копіювання аудіо + Постобробка і ACW + Аналітика і QA | Автозаповнення картки + контроль якості → ROI < 1 місяць |
| **Ітерація 2** | Підказки оператору (Real-time Copilot) | AI-підказки під час дзвінка → AHT -10–15% |
| **Ітерація 3** | Голосовий асистент (Pre-Call AI) | AI розмовляє з клієнтом до оператора → після підтвердження ROI Ітерацій 1–2 |

## Як блоки пов'язані між собою

```
Клієнт дзвонить
      │
      ▼
Блок 2: AI-асистент до оператора (Pre-Call AI) [Після P1]
      │ передає контекст розмови
      ▼
Блок 1: Копіювання аудіо дзвінка
      │
      ├──────────────────────────────────────┐
      ▼                                      ▼
Блок 4: Підказки під час дзвінка      Блок 3: Запис і розпізнавання
(Real-time Copilot → РМ КЦ)          (під час дзвінка)
                                             │
                                             ▼ після завершення дзвінка
                                      Заповнення картки РМ КЦ (ACW)
                                             │
                                             ▼
                                      Блок 5: Аналітика і QA
                                      (тренди, скоринг, coaching)
```

## Відкриті питання перед стартом пілоту

1. **Створити GCP акаунт** — 1 день реєстрація + 3–5 днів на квоти (потрібен для Блоків 3, 4, 5 на Google)
2. **POC Ender Turing** — протестувати якість аналізу на реальних дзвінках НП (uk-UA)
3. **Підтвердити API РМ КЦ** — чи є ендпоінт для автоматичного заповнення картки після дзвінка
4. **ElevenLabs scoping call** — уточнити Enterprise pricing і SLA при 30K дзвінків/день (якщо Блок 2 в планах)
5. **Перевірити Cisco UCCE версію** — підтвердити що SIPREC media-forking включений і налаштований
